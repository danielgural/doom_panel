# Game in FiftyOne!

![GAME](https://github.com/danielgural/doom_panel/blob/main/assets/game.gif)

This panel is adds the ability to play the hit classics in your fiftyone app! 

## Installation
```
fiftyone plugins download https://github.com/danielgural/game_panel
```

## DOOM
![DOOM](https://github.com/danielgural/doom_panel/blob/main/assets/doom.gif)
Controls:
Move: UP, DOWN, LEFT, RIGHT 
Use: W 
Fire: S 
Speed on: SPACE 
Strafe on: ALT 
Strafe: A, D Change Weapon: 1, 2, 3, 4, 5, 6, 7

## How it works!

This panel wraps an iframe aroun the urls, like https://thedoggybrad.github.io/doom_on_js-dos/. This allows you to interact with the game like you would in your browser! A great example of the powerful ways to use iframes and panels!
